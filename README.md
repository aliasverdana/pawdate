# Pawdate Stockholm

Find compatible dog playdates in Stockholm.

## Stack

- Next.js (App Router) + TypeScript
- Radix UI Themes

## Local dev

```bash
npm install
npm run dev
```

## Roadmap (v0)

- Dog profiles (name, size, energy, play style, neighborhood)
- Browse and filters
- Playdate requests (request, accept, decline)
- Safety basics (block, report)
